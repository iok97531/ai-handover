# Scope 개인정보처리방침

**최종 수정일:** 2026년 4월 1일

**시행일:** 2026년 4월 1일

---

## 1. 개요

Scope(이하 "본 소프트웨어")는 사용자의 로컬 컴퓨터에서 실행되는 데스크톱 애플리케이션입니다. 본 방침은 본 소프트웨어가 사용자의 데이터를 어떻게 처리하는지 설명합니다.

---

## 2. 수집하는 개인정보

**본 소프트웨어는 개인정보를 수집하지 않습니다.**

- 회원가입이 없습니다.
- 이메일, 이름, 전화번호 등 개인 식별 정보를 요구하지 않습니다.
- 사용자 행동 분석(텔레메트리), 추적 도구, 쿠키를 사용하지 않습니다.
- 사용 통계를 외부 서버로 전송하지 않습니다.

---

## 3. 데이터 저장 위치

모든 데이터는 **사용자의 로컬 컴퓨터에만** 저장되며, 외부 서버로 전송되지 않습니다.

| 데이터 | 저장 위치 | 설명 |
|--------|-----------|------|
| 프로젝트 정보, 채팅 기록, 설정 | `{userData}/data/app.db` (SQLite) | 프로젝트 목록, AI 대화 내용, 앱 설정 |
| 벡터 임베딩 데이터 | `{userData}/data/chroma/` (ChromaDB) | 코드/문서를 AI가 검색할 수 있도록 변환한 데이터 |
| AI API 키 | `{userData}/data/app.db` 내 암호화 저장 | Fernet(AES) 대칭키 암호화 적용 |

> **`{userData}` 경로:**
> - Windows: `%APPDATA%\scope\data\`
> - macOS: `~/Library/Application Support/scope/data/`
> - Linux: `~/.config/scope/data/`

---

## 4. 외부 전송되는 데이터

본 소프트웨어는 AI 기능을 위해 **사용자가 직접 설정한 외부 AI API**에 데이터를 전송합니다.

### 전송 대상 및 내용

| 전송 대상 | 전송 내용 | 시점 |
|-----------|-----------|------|
| OpenAI API 또는 Anthropic API | 코드/문서의 텍스트 조각 (임베딩용) | 프로젝트 인덱싱 시 |
| OpenAI API 또는 Anthropic API | 사용자 질문 + 관련 텍스트 조각 (답변 생성용) | AI 질의응답 시 |

### 중요 사항

- **사용자가 직접 API 키를 입력한 경우에만** 외부 전송이 발생합니다.
- 전송되는 데이터는 사용자의 코드/문서 중 **검색에 필요한 일부 텍스트 조각**입니다.
- Anthropic Claude 사용 시, 임베딩은 **로컬 모델**(`all-MiniLM-L6-v2`)로 처리되므로 임베딩 목적의 외부 전송은 발생하지 않습니다.
- 외부 AI 서비스 제공자의 데이터 처리 정책은 해당 서비스의 개인정보처리방침을 참조하시기 바랍니다.
  - [OpenAI 개인정보처리방침](https://openai.com/policies/privacy-policy)
  - [Anthropic 개인정보처리방침](https://www.anthropic.com/privacy)

---

## 5. API 키 보안

- 사용자가 입력한 AI API 키는 `cryptography` 라이브러리의 **Fernet(AES 128-bit CBC)** 방식으로 암호화하여 로컬 SQLite 데이터베이스에 저장합니다.
- 암호화 키는 사용자의 로컬 환경에 저장되며, 외부로 전송되지 않습니다.
- API 키 원문은 메모리에서만 복호화되어 사용되며, 로그 파일 등에 기록되지 않습니다.

---

## 6. 데이터 삭제

본 소프트웨어의 모든 데이터를 삭제하려면 다음 폴더를 삭제하십시오.

**Windows:**
```
%APPDATA%\scope\data\
```

**macOS:**
```
~/Library/Application Support/scope/data/
```

**Linux:**
```
~/.config/scope/data/
```

앱 내에서 개별 프로젝트를 삭제하면 해당 프로젝트의 채팅 기록과 벡터 데이터도 함께 삭제됩니다.
