## 1. 사업자 등록

| 항목 | 사업자 필요 여부 |
|------|-----------------|
| 무료 앱 배포 | 불필요 |
| 코드 사이닝 인증서 구매 | **필요** (법인 또는 개인사업자) |
| Stripe 결제 연동 (Phase 5) | **필요** |

**추천:** 개인사업자 (간이과세자)로 시작

- 향후 매출 8,000만원 초과 시 일반과세자 전환

**필요 서류:**
- [ ] 신분증

## 2. 앱 아이콘 제작

**필수 — 이것 없이는 인스톨러 빌드 불가**

| 필요 파일 | 크기 | 용도 |
|-----------|------|------|
| `build/icon.ico` | 256x256 (멀티사이즈) | Windows 인스톨러, 작업표시줄 |
| `build/icon.png` | 512x512 | 일반 용도, 랜딩페이지 |
| 트레이 아이콘 | 16x16, 32x32 | 시스템 트레이 |

**방법 (택 1):**
- 직접 디자인 (Figma 등)
- AI 생성 (DALL-E, Midjourney) → .ico 변환
- 외주 (크몽, 숨고 등 — 5~15만원, 3~7일)

**변환 도구:** https://convertio.co/png-ico/ (PNG → ICO 멀티사이즈)

## 3. 코드 사이닝 인증서

**없으면 Windows SmartScreen이 "알 수 없는 게시자" 경고를 띄움 → 사용자 이탈 주요 원인**

| 옵션 | 비용 | 특징 |
|------|------|------|
| **OV 인증서** (권장) | $100~200/년 | 개인/법인 가능, SmartScreen 경고 감소 (완전 제거는 아님) |
| **EV 인증서** | $300~500/년 | 법인 필수, SmartScreen 경고 완전 제거, 발급 1~2주 |
| **인증서 없이 배포** | 무료 | SmartScreen 경고 뜨지만 "추가 정보" → "실행" 가능 |

**Phase 1 권장:** OV 인증서 또는 인증서 없이 시작 (얼리어답터는 경고 무시 가능)

**발급처:**
- Sectigo (구 Comodo) — 가장 대중적
- DigiCert — 기업용
- SSL.com — 개인 개발자 친화적

**절차:**
- [ ] 사업자등록증 (또는 신분증)
- [ ] 발급처에서 구매 + 신원 확인 (1~5일)
- [ ] .pfx 파일 수령
- [ ] electron-builder 설정에 인증서 경로 추가

## 5. 배포 채널 결정

**.exe 설치 파일을 어디서 제공할 것인가**

| 옵션 | 장점 | 단점 |
|------|------|------|
| **GitHub Releases** (권장) | 무료, 자동화 쉬움, 개발자 친숙 | 비개발자에게 낯설 수 있음 |
| **랜딩페이지 직접 호스팅** | UX 깔끔 | 대역폭 비용, CDN 필요 |
| **두 곳 모두** | 최적 | 관리 포인트 증가 |

**Phase 1 권장:** GitHub Releases + 랜딩페이지에서 링크

**결정 필요:**
- [ ] GitHub 리포를 public으로 전환할 것인가? (오픈소스 여부)
- [ ] private이면 별도 다운로드 서버 필요

## 6. 빌드 검증 (E2E 테스트)

**반드시 본인 환경에서 직접 확인해야 함**

```bash
# 1. 백엔드 빌드
npm run build:backend

# 2. 전체 빌드 (Electron + 인스톨러)
npm run build:full

# 3. 생성된 인스톨러 실행
#    out/ 또는 dist/ 폴더에서 .exe 찾기

# 4. 설치 후 확인 체크리스트:
```

- [ ] 인스톨러가 정상 실행되는가
- [ ] 설치 후 앱이 실행되는가
- [ ] 백엔드(Python)가 자동 시작되는가
- [ ] API 키 입력 후 프로젝트 등록이 되는가
- [ ] 인덱싱이 완료되는가
- [ ] AI Q&A가 정상 동작하는가
- [ ] 앱 종료 시 백엔드 프로세스도 종료되는가
- [ ] 앱 재실행 시 기존 데이터가 유지되는가
- [ ] 다른 PC에서도 동일하게 동작하는가 (가능하면 클린 환경)
