# Scope — 제품 로드맵

사업 모델: Free → Pro → Enterprise 순서로 단계적 확장.
각 Phase는 이전 Phase의 사용자 기반과 피드백을 기반으로 진행한다.

---

## Phase 1 — Free 출시
> **목표:** 사용자 확보 및 피드백 수집

- [x] 핵심 기능 완성 (코드 인덱싱, AI Q&A, 인수인계 4종)
- [x] 랜딩페이지 제작 (getscope.dev)
- [ ] PyInstaller 번들링 — Python 설치 없이 바로 실행 가능
- [ ] Windows 배포 (exe 설치 파일)
- [ ] Free 플랜 공개 배포 (프로젝트 1개, 본인 API 키 사용)

---

## Phase 2 — Pro 출시
> **목표:** 첫 유료 수익, 팀 협업 기능으로 구독 전환

- [ ] 라이선스 서버 구축
  - 온라인 구독 체크 + 30일 로컬 캐시
  - 구독 해지 시 30일 후 자동 만료
- [ ] Pro 기능 개발
  - 무제한 프로젝트
  - 인수인계 문서 공유 링크
  - 팀 채팅 히스토리 공유
- [ ] Pro 결제 연동 (Stripe)
- [ ] macOS 지원

**요금:** 월 $19 / 인

---

## Phase 3 — Enterprise 출시
> **목표:** 고단가 B2B 계약 (건당 수천만 원)
> **세일즈 포인트:** "소스코드가 회사 밖으로 한 줄도 나가지 않는다"

- [ ] 로컬 LLM 프로바이더 연동
  - Ollama, vLLM, LM Studio
  - Azure OpenAI Private Endpoint
  - 설정 UI에서 커스텀 엔드포인트 입력
- [ ] 완전 오프라인 모드 (로컬 LLM + 로컬 임베딩)
- [ ] 오프라인 라이선스 키 시스템
  - JWT + RSA/ECDSA 서명, 앱에 공개키 내장
  - 머신 ID 기반 발급, 만료일 포함
  - 갱신: 만료 30일 전 경고 → 이메일로 새 키 발급
- [ ] 온프레미스 설치 패키지
- [ ] 보안 감사 대응 문서

**요금:** 시트 라이선스 (카피당 × 연간 계약), 협의

**주요 타겟:**
| 타겟 | 이유 |
|------|------|
| 금융·의료·공공기관 | 외부 API 사용 규제, 데이터 주권 요건 |
| 대형 SI·IT 서비스사 | 고객사 코드 외부 유출 금지 계약 |
| 제조·방산 | 인터넷 차단 환경, 사내 LLM 운영 중 |
