// 작동 방식

3단계로 끝납니다

설치하고, 폴더를 열고, 질문하면 됩니다.

01

폴더 선택 & 인덱싱

프로젝트 폴더를 선택하면 코드를 자동으로 분석해 로컬에 인덱싱합니다. 코드는 외부로 나가지 않습니다.

02

인수인계 형식 선택

전체 요약, 개발자 온보딩, 운영·배포 가이드, 이슈 현황 중 필요한 형식을 선택합니다.

03

AI와 대화로 파악

AI가 코드 기반으로 설명해주고, 추가 질문도 자유롭게 할 수 있습니다.

// 기능

필요한 건 다 있습니다

🔐

완전 로컬 처리

코드가 우리 서버로 전송되지 않습니다. 인덱싱과 검색은 모두 로컬에서 처리됩니다.

🤖

4가지 인수인계 형식

전체 요약 / 개발자 온보딩 / 운영·배포 가이드 / 이슈·기술 부채 현황. 목적에 맞게 선택하세요.

💬

대화형 Q&A

인수인계 후에도 "이 함수가 뭐해요?", "왜 이렇게 짰어요?" 같은 질문을 자유롭게 할 수 있습니다.

👁️

파일 변경 자동 감지

코드가 바뀌면 자동으로 인덱스를 업데이트합니다. 항상 최신 코드 기준으로 답변합니다.

🔌

OpenAI & Claude 지원

OpenAI GPT-4o, Claude Sonnet 등 주요 AI 모델을 지원합니다. 본인 API 키를 사용합니다.

📂

다양한 언어 지원

Python, TypeScript, Java, Go, Rust 등 40+ 언어와 파일 형식을 지원합니다.

// 보안

소스코드가 회사 밖으로
한 줄도 나가지 않습니다

코드 인덱싱과 검색은 100% 로컬에서 처리됩니다. AI 질의 시에는 선택한 API(OpenAI/Claude)로만 전송되며, 우리 서버는 코드를 저장하거나 열람하지 않습니다.

✓ 코드 서버 저장 없음 ✓ 로컬 ChromaDB 벡터 저장 ✓ 암호화된 API 키 저장 ⊕ Enterprise: 완전 오프라인 지원 ⊕ 사내 LLM 서버 연동
